# TEST-001: CLI Calculator Tests

## Task Description
The `src/calculator.py` file contains a CLI calculator with basic operations (add, subtract, multiply, divide). The `tests/` directory is empty - you need to write the tests.

## Requirements
- Write at least 8 tests in `tests/test_calculator.py`
- Achieve at least 80% code coverage
- Test all four operations
- Test edge cases (zero, negative numbers, division by zero)

## Files
- `src/calculator.py` - The calculator implementation (do not modify)
- `tests/` - Directory where you add tests
- `check.sh` - Runs pytest with coverage and requires >= 8 tests and >= 80% coverage
